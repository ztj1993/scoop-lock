# WinSCP - 文件传输工具使用指南

## 简介

WinSCP 是一款 Windows 环境下开源的文件传输客户端，支持多种文件传输协议。

它可以安全地在本地计算机和远程服务器之间复制文件，是系统管理员和开发人员的必备工具。

WinSCP 提供图形界面和命令行两种使用方式，支持中文界面，操作简单直观。

| 可执行文件 | WinSCP.exe, WinSCP-cli.exe |

### 支持的协议

WinSCP 支持以下文件传输协议：

- **SFTP** - 基于 SSH 的文件传输协议（推荐）
- **SCP** - 安全复制协议
- **FTP** - 文件传输协议
- **FTPS** - FTP over SSL/TLS
- **WebDAV** - Web 分布式创作和版本控制
- **S3** - Amazon S3 云存储协议

## 常用操作

### 建立连接

#### 图形界面连接

1. 启动 WinSCP
2. 在登录对话框中填写连接信息：
   - 文件协议：选择 SFTP（推荐）
   - 主机名：输入服务器地址
   - 端口：默认 22（SFTP）或 21（FTP）
   - 用户名：输入登录用户名
   - 密码：输入登录密码
3. 点击"登录"按钮
4. 首次连接会提示保存服务器指纹

#### 快速连接

1. 点击工具栏的"新建会话"按钮
2. 或使用快捷键 Ctrl+N
3. 输入连接信息
4. 点击"保存"可保存为站点

#### 命令行连接

使用 WinSCP-cli.exe 连接：
```
WinSCP-cli.exe /command "open sftp://user:password@hostname/"
```

### 文件传输操作

#### 上传文件

方法一：拖放上传
1. 在左侧面板（本地）选择文件
2. 拖拽到右侧面板（远程）
3. 等待传输完成

方法二：使用按钮
1. 选择要上传的文件
2. 点击工具栏的"上传"按钮
3. 或按 F5 键
4. 确认传输设置
5. 点击"确定"

方法三：右键菜单
1. 右键点击文件
2. 选择"上传"
3. 选择目标目录
4. 确认传输

#### 下载文件

方法一：拖放下载
1. 在右侧面板（远程）选择文件
2. 拖拽到左侧面板（本地）
3. 等待传输完成

方法二：使用按钮
1. 选择要下载的文件
2. 点击工具栏的"下载"按钮
3. 或按 F5 键
4. 选择本地目标目录
5. 确认传输

#### 批量传输

1. 按住 Ctrl 键选择多个文件
2. 或按 Ctrl+A 选择全部文件
3. 执行上传或下载操作
4. 所有选中的文件将被传输

### 文件管理操作

#### 创建文件夹

1. 在目标面板右键点击空白处
2. 选择"新建" → "文件夹"
3. 输入文件夹名称
4. 按 Enter 确认

#### 删除文件

1. 选择要删除的文件
2. 按 Delete 键
3. 或右键选择"删除"
4. 确认删除操作

#### 重命名文件

1. 选择文件
2. 按 F2 键
3. 或右键选择"重命名"
4. 输入新名称
5. 按 Enter 确认

#### 修改权限

1. 选择文件
2. 按 F9 键
3. 或右键选择"属性"
4. 修改权限设置（如 755, 644）
5. 点击"确定"

### 远程编辑文件

#### 使用内置编辑器

1. 双击远程文件
2. 或右键选择"编辑"
3. 内置编辑器打开文件
4. 修改后按 Ctrl+S 保存
5. 文件自动上传到服务器

#### 使用外部编辑器

1. 右键点击文件
2. 选择"编辑" → "使用外部编辑器"
3. 文件在外部编辑器中打开
4. 保存后自动检测变化并上传

#### 编辑器设置

在"选项" → "编辑器"中配置：
- 默认编辑器选择
- 自动保存设置
- 编码格式设置
- 语法高亮配置

### 目录同步

#### 启动同步

1. 点击菜单 "Commands" → "Synchronize"
2. 或按 Ctrl+S
3. 设置同步参数：
   - 同步方向（本地到远程/远程到本地/双向）
   - 同步模式（完整/智能）
   - 文件过滤规则
4. 点击"预览"查看同步计划
5. 点击"确定"开始同步

#### 同步模式说明

| 模式 | 说明 |
|------|------|
| 完整 | 同步所有文件，包括已存在的 |
| 智能 | 只同步有变化的文件 |
| 增量 | 只传输新增和修改的文件 |

#### 同步规则配置

文件过滤：
- 指定要包含的文件类型
- 指定要排除的文件类型
- 使用通配符匹配

同步选项：
- 包含子文件夹
- 删除目标中多余的文件
- 同步文件权限
- 同步时间戳

### 会话管理

#### 保存会话

1. 在登录对话框中输入连接信息
2. 点击"保存"按钮
3. 输入会话名称
4. 选择保存位置（个人/共享）
5. 点击"确定"

#### 管理站点

1. 点击菜单 "Session" → "Sites" → "Site Manager"
2. 在站点管理器中：
   - 创建文件夹分组
   - 编辑现有站点
   - 删除不需要的站点
   - 导入/导出站点配置

#### 会话导入导出

导出：
1. 点击 "Tools" → "Export/Backup"
2. 选择要导出的内容
3. 指定导出文件路径
4. 设置密码保护（可选）

导入：
1. 点击 "Tools" → "Import/Restore"
2. 选择导入文件
3. 选择导入选项
4. 完成导入

## 命令行使用 (WinSCP-cli)

### 基本语法

```
WinSCP-cli.exe /command "command1" "command2" ...
```

### 常用命令行操作

#### 上传文件

```batch
WinSCP-cli.exe /command ^
    "open sftp://user:password@host/" ^
    "put C:\local\file.txt /remote/path/" ^
    "exit"
```

#### 下载文件

```batch
WinSCP-cli.exe /command ^
    "open sftp://user:password@host/" ^
    "get /remote/file.txt C:\local\path\" ^
    "exit"
```

#### 同步目录

```batch
WinSCP-cli.exe /command ^
    "open sftp://user:password@host/" ^
    "synchronize remote C:\local\path /remote/path" ^
    "exit"
```

#### 使用脚本文件

创建脚本文件 `upload.txt`：
```
open sftp://user:password@host/
put C:\data\* /upload/
exit
```

执行脚本：
```
WinSCP-cli.exe /script=upload.txt
```

### 命令行参数详解

| 参数 | 说明 |
|------|------|
| /command | 执行指定的命令 |
| /script | 执行脚本文件 |
| /log | 记录日志到文件 |
| /ini | 指定配置文件路径 |
| /console | 使用控制台模式 |
| /passive | 使用被动模式（FTP） |

### 脚本命令参考

| 命令 | 说明 |
|------|------|
| open | 打开连接 |
| close | 关闭连接 |
| exit | 退出程序 |
| put | 上传文件 |
| get | 下载文件 |
| synchronize | 同步目录 |
| cd | 切换远程目录 |
| lcd | 切换本地目录 |
| ls | 列出远程文件 |
| mkdir | 创建远程目录 |
| rm | 删除远程文件 |
| chmod | 修改文件权限 |

3. 选择 "Languages"
4. 在语言列表中选择"中文（简体）"
5. 点击"确定"
6. 重启 WinSCP 生效

### 文件名编码设置

处理中文文件名乱码：

1. 打开会话设置
2. 进入 "Environment" → "SCP/Shell"
3. 设置编码为 UTF-8
4. 或在 "Advanced" → "Other" 中设置编码

### 常见编码问题

| 日志显示异常 | 配置日志编码 |

## 高级配置

### 传输设置优化

#### 传输模式

自动模式：
- WinSCP 自动检测文件类型
- 文本文件使用 ASCII 模式
- 二进制文件使用二进制模式

手动设置：
1. 打开传输设置对话框
2. 选择传输模式
3. ASCII - 用于文本文件
4. 二进制 - 用于所有文件（推荐）

#### 传输选项配置

在传输对话框中可以设置：
- 传输模式（自动/ASCII/二进制）
- 保留时间戳
- 保留权限
- 保留所有权
- 传输后操作

### SSH 密钥配置

#### 导入 PuTTY 私钥

1. 打开会话设置
2. 进入 "SSH" → "Authentication"
3. 点击"..."选择私钥文件
4. 支持 .ppk 格式的 PuTTY 密钥

#### 使用 OpenSSH 密钥

1. 如有 OpenSSH 格式密钥，需要先转换
2. 使用 PuTTYgen 工具转换格式
3. 加载私钥并保存为 .ppk 格式
4. 在 WinSCP 中使用转换后的密钥

### 自动化任务

#### 创建自动化脚本

1. 使用记事本创建脚本文件
2. 编写 WinSCP 命令
3. 保存为 .txt 文件
4. 使用 WinSCP-cli.exe 执行

#### 定时执行任务

使用 Windows 任务计划程序：

1. 打开"任务计划程序"
2. 创建新任务
3. 设置触发器（时间/事件）
4. 设置操作：
   - 程序：WinSCP-cli.exe
   - 参数：/script=your_script.txt
5. 保存并测试任务

### 日志配置

#### 启用日志记录

1. 打开 "Options" → "Preferences"
2. 进入 "Logging"
3. 启用日志记录
4. 设置日志文件路径
5. 选择日志级别

#### 日志级别说明

| 级别 | 记录内容 |
|------|---------|
| Normal | 基本操作日志 |
| Debug | 详细调试信息 |
| Trace | 完整跟踪信息 |

## 常见问题

### 1. 连接被拒绝

可能原因及解决：
1. 检查服务器地址和端口是否正确
2. 确认 SSH 服务正在运行
3. 检查防火墙设置
4. 确认用户名密码正确
5. 检查服务器 IP 白名单

### 2. 传输中断

解决方案：
1. 检查网络连接稳定性
2. 启用断点续传功能
3. 使用二进制传输模式
4. 减少并发传输数量
5. 增加超时时间设置

### 3. 权限被拒绝

排查步骤：
1. 检查文件/目录权限
2. 确认用户有足够的权限
3. 联系服务器管理员
4. 尝试使用 sudo 提权（如允许）

### 4. 中文文件名乱码

解决方法：
1. 在会话设置中启用 UTF-8 编码
2. 使用 UTF-8 编码的终端
3. 确认服务器 locale 设置正确
4. 尝试不同的编码设置

### 5. 传输速度慢

优化建议：
1. 使用 SFTP 而非 SCP
2. 启用压缩传输
3. 使用二进制模式
4. 减少目录列表刷新
5. 批量传输文件

### 6. 如何跳过主机密钥验证？

不建议跳过，但如需临时跳过：
1. 在登录时选择"是"接受未知主机密钥
2. 或在站点设置中配置接受策略

### 7. 如何设置默认本地目录？

配置方法：
1. 打开会话属性
2. 进入 "Environment" → "Directories"
3. 设置 "Local directory" 默认路径
4. 保存设置

### 8. 如何传输隐藏文件？

显示和传输隐藏文件：
1. 点击菜单 "Options" → "Preferences"
2. 进入 "Panels"
3. 勾选"Show hidden files"
4. 刷新目录列表

### 9. 如何批量重命名文件？

使用通配符重命名：
1. 选择多个文件
2. 右键选择"批量重命名"
3. 使用通配符模式
4. 预览结果
5. 确认重命名

### 10. 如何恢复意外删除的文件？

WinSCP 本身不提供回收站功能：
1. 检查服务器是否有回收站
2. 从备份中恢复
3. 联系服务器管理员
4. 重要文件建议先备份再删除
