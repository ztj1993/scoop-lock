# PicPick v7.5.0 使用指南

## 简介

PicPick 是一款功能全面的屏幕捕获工具和图像编辑器，为用户提供了直观易用的界面和丰富的功能。作为 Shareware 软件，PicPick 在免费版本中已经提供了强大的屏幕捕获和图像编辑能力。

该软件集成了屏幕截图、图像编辑、颜色拾取、像素标尺、量角器、十字线等多种实用工具，是设计师、开发者和普通用户的理想选择。PicPick 的界面设计简洁直观，即使是初次使用的用户也能快速上手。

PicPick 支持多种屏幕捕获模式，包括全屏截图、窗口截图、区域截图、滚动窗口截图等。捕获的图像可以直接在内置编辑器中进行编辑和标注，无需使用其他图像编辑软件。

本指南将详细介绍 PicPick 的各项功能和使用方法，帮助用户充分利用这款优秀的屏幕捕获和图像编辑工具。

## 主要功能

### 屏幕捕获功能

PicPick 提供了多种屏幕捕获模式，满足不同场景的需求。全屏捕获可以捕获整个屏幕的内容，窗口捕获可以捕获特定窗口，区域捕获允许用户自定义捕获区域。

滚动窗口捕获是 PicPick 的特色功能之一，能够捕获整个网页或文档的内容，即使内容超出了当前屏幕显示范围。这对于制作教程、文档截图等场景非常有用。

支持延时捕获功能，用户可以设置延迟时间后再进行捕获。这对于捕获菜单、下拉列表、工具提示等需要特定操作才能显示的内容非常实用。

提供了屏幕录制功能，可以录制屏幕操作过程并保存为视频文件。这对于制作教程、演示操作步骤等场景非常有帮助。

### 图像编辑功能

PicPick 内置了功能强大的图像编辑器，支持常见的图像编辑操作。包括裁剪、旋转、调整大小、调整亮度和对比度等基本功能。

提供了丰富的绘图工具，包括画笔、铅笔、形状、文字、箭头等。用户可以在截图上添加标注、说明、高亮等内容。

支持图层功能，允许用户在不同的图层上进行编辑，便于管理和修改复杂的图像内容。这大大提高了编辑的灵活性和效率。

提供了多种滤镜和特效，包括模糊、锐化、浮雕、马赛克等。这些效果可以用于美化图像或隐藏敏感信息。

### 颜色拾取工具

PicPick 包含了专业的颜色拾取工具，能够精确获取屏幕上任意位置的颜色值。支持多种颜色格式，包括 RGB、HEX、HSL、HSB 等。

提供了颜色历史记录功能，可以保存和管理之前拾取的颜色。这对于需要重复使用特定颜色的设计工作非常有帮助。

支持颜色调板功能，用户可以创建和保存自己的颜色调板。这有助于保持设计作品中颜色的一致性。

### 测量工具

PicPick 提供了像素标尺工具，可以精确测量屏幕上元素的尺寸。支持水平和垂直方向的测量，单位可以是像素、英寸、厘米等。

量角器工具可以测量角度，对于需要精确角度的设计工作非常有用。十字线工具可以显示精确的坐标位置，便于精确定位。

提供了屏幕取色器功能，可以实时显示鼠标位置的颜色值和坐标信息。这对于设计工作和开发调试都非常有帮助。

### 文件管理功能

PicPick 支持多种图像格式的保存和打开，包括 PNG、JPEG、BMP、GIF、TIFF 等常见格式。用户可以根据需要选择合适的格式和质量设置。

提供了批量处理功能，可以同时对多个图像文件进行格式转换、重命名、调整大小等操作。这大大提高了处理大量图像文件的效率。

支持图像上传功能，可以将图像直接上传到 FTP 服务器、云存储或社交媒体平台。这简化了图像分享和发布的流程。

## 常用命令/操作

### 安装和启动

通过 Scoop 安装 PicPick：
```bash
scoop install picpick
```

启动 PicPick：
```bash
# 直接运行可执行文件
picpick.exe
```

查看版本信息：
```bash
picpick.exe --version
```

### 基本截图操作

**全屏截图**：
- 使用快捷键 `Print Screen` 或 `Ctrl + Print Screen`
- 在系统托盘中右键点击 PicPick 图标，选择"全屏截图"

**窗口截图**：
- 使用快捷键 `Alt + Print Screen`
- 在系统托盘中右键点击 PicPick 图标，选择"窗口截图"

**区域截图**：
- 使用快捷键 `Ctrl + Shift + Print Screen`
- 在系统托盘中右键点击 PicPick 图标，选择"区域截图"
- 拖动鼠标选择要捕获的区域

**滚动窗口截图**：
- 使用快捷键 `Ctrl + Alt + Print Screen`
- 在系统托盘中右键点击 PicPick 图标，选择"滚动窗口截图"
- 点击要捕获的窗口，PicPick 会自动滚动并捕获整个内容

### 延时截图设置

**设置延时时间**：
1. 打开 PicPick 主界面
2. 进入"设置"菜单
3. 选择"捕获"选项卡
4. 设置"延时捕获"时间（秒）

**使用延时截图**：
1. 使用快捷键或菜单启动延时截图
2. 在倒计时结束前准备好要捕获的内容
3. PicPick 会在倒计时结束后自动截图

### 图像编辑操作

**基本编辑**：
1. 打开图像文件或截图
2. 使用工具栏中的编辑工具
3. 进行裁剪、旋转、调整大小等操作
4. 保存修改后的图像

**添加标注**：
1. 使用"文字"工具添加文字说明
2. 使用"箭头"工具指向特定区域
3. 使用"形状"工具绘制矩形、圆形等
4. 使用"高亮"工具突出显示重要部分

**应用滤镜**：
1. 选择要应用滤镜的图像或区域
2. 在"滤镜"菜单中选择所需效果
3. 调整滤镜参数
4. 预览并应用滤镜效果

### 颜色拾取操作

**拾取颜色**：
1. 启动颜色拾取工具
2. 将鼠标移动到要拾取颜色的位置
3. 点击鼠标左键拾取颜色
4. 颜色值会显示在颜色面板中

**复制颜色值**：
1. 拾取颜色后，在颜色面板中选择颜色格式
2. 右键点击颜色值，选择"复制"
3. 在需要的地方粘贴颜色值

**管理颜色历史**：
1. 在颜色面板中查看颜色历史记录
2. 右键点击历史记录中的颜色
3. 选择"保存到调板"或"删除"

### 测量工具使用

**使用像素标尺**：
1. 启动像素标尺工具
2. 将标尺放置在要测量的位置
3. 读取标尺上的数值
4. 支持水平和垂直测量

**使用量角器**：
1. 启动量角器工具
2. 将量角器中心点放置在角的顶点
3. 调整量角器的角度
4. 读取角度值

**使用十字线**：
1. 启动十字线工具
2. 将十字线移动到目标位置
3. 查看坐标值和颜色信息
4. 支持坐标单位切换

**上传图像**：

4. 开始上传并获取分享链接

## 常见问题

### 截图无法保存

**问题描述**：截图后无法保存到指定位置。

**解决方案**：
1. 检查保存路径是否存在且有写入权限
2. 确认磁盘空间是否充足
3. 检查文件名是否包含非法字符
4. 尝试更改保存格式或质量设置

```bash
# 检查目录权限
icacls "C:\Users\Username\Pictures" /grant Users:F
```

### 滚动截图不完整

**问题描述**：滚动窗口截图只捕获了部分内容。

**解决方案**：
1. 确保目标窗口支持滚动
2. 检查窗口是否有固定的头部或底部区域
3. 调整滚动速度设置
4. 尝试使用不同的捕获模式

**检查设置**：
1. 打开"设置" -> "捕获"
2. 调整"滚动延迟"参数
3. 启用"自动检测滚动区域"
4. 测试不同的滚动窗口

### 快捷键冲突

**问题描述**：PicPick 快捷键与其他软件冲突。

**解决方案**：
1. 打开"设置" -> "快捷键"
2. 修改冲突的快捷键
3. 选择不常用的快捷键组合
4. 或者禁用冲突软件的快捷键

**推荐快捷键**：
- 全屏截图：`Ctrl + Shift + F`
- 窗口截图：`Ctrl + Shift + W`
- 区域截图：`Ctrl + Shift + R`
- 滚动截图：`Ctrl + Shift + S`

### 编辑器性能问题

**问题描述**：图像编辑器响应缓慢或卡顿。

**解决方案**：
1. 减小图像尺寸或分辨率
2. 关闭不必要的图层和效果
3. 增加系统内存
4. 更新显卡驱动程序

**优化设置**：
1. 在"设置" -> "性能"中调整参数
2. 降低图像预览质量
3. 禁用实时预览效果
4. 定期清理临时文件

### 颜色拾取不准确

**问题描述**：拾取的颜色值与实际颜色不符。

**解决方案**：
1. 确保显示器颜色配置正确
2. 检查是否有颜色滤镜或护眼模式启用
3. 校准显示器颜色
4. 使用标准颜色配置文件

**校准步骤**：
1. 打开 Windows 显示设置
2. 运行显示器颜色校准工具
3. 按照向导完成校准
4. 保存颜色配置文件

### 文件格式兼容性

**问题描述**：某些图像格式无法打开或保存。

**解决方案**：
1. 检查 PicPick 是否支持该格式
2. 安装必要的编解码器
3. 转换为支持的格式
4. 更新 PicPick 到最新版本

**支持格式**：
- **输入**：PNG, JPEG, BMP, GIF, TIFF, ICO, CUR, ANI
- **输出**：PNG, JPEG, BMP, GIF, TIFF, PDF

### 启动和运行问题

**问题描述**：PicPick 无法启动或频繁崩溃。

**解决方案**：
1. 以管理员身份运行 PicPick
2. 检查系统兼容性要求
3. 更新 .NET Framework 和 Visual C++ 运行时
4. 清理临时文件和缓存

**系统要求**：
- Windows 7/8/10/11
- .NET Framework 4.5 或更高版本
- 2GB 内存（推荐 4GB）
- 100MB 可用磁盘空间

### 图像格式说明

**PNG 格式**：
- 支持透明背景
- 无损压缩
- 文件大小较大
- 适合截图和图标

**JPEG 格式**：
- 不支持透明背景
- 有损压缩
- 文件大小较小
- 适合照片和复杂图像

**BMP 格式**：
- 无压缩
- 文件大小很大
- 兼容性好
- 适合特殊需求

**GIF 格式**：
- 支持动画
- 256 色限制
- 文件大小较小
- 适合简单动画

**TIFF 格式**：
- 支持多页
- 无损压缩
- 文件大小较大
- 适合专业印刷

### 颜色代码格式

**RGB 格式**：
- `rgb(255, 0, 0)` - 红色
- `rgb(0, 255, 0)` - 绿色
- `rgb(0, 0, 255)` - 蓝色

**HEX 格式**：
- `#FF0000` - 红色
- `#00FF00` - 绿色
- `#0000FF` - 蓝色

**HSL 格式**：
- `hsl(0, 100%, 50%)` - 红色
- `hsl(120, 100%, 50%)` - 绿色

### 性能优化建议

**内存优化**：
1. 关闭不需要的编辑器标签
2. 定期清理剪贴板历史
3. 减少同时打开的图像数量
4. 使用适当的图像分辨率

**存储优化**：
1. 选择合适的图像格式
2. 调整压缩质量参数
3. 定期清理临时文件
4. 使用云存储备份重要文件

**网络优化**：
1. 优化上传图像大小
2. 使用适当的网络设置
3. 配置代理服务器（如需要）
4. 监控网络连接状态
