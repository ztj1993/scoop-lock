# WindTerm - 高级终端客户端使用指南

## 简介

WindTerm 是一款功能强大的终端客户端，支持 SSH、SFTP、Shell、Telnet 和 Serial 等多种连接协议。

它提供了现代化的用户界面和丰富的功能，适合系统管理员、网络工程师和开发人员使用。

WindTerm 以高性能和低资源占用著称，是 PuTTY、Xshell 等传统终端工具的优秀替代品。

## 常用操作

### 创建 SSH 连接

#### 步骤一：新建会话

1. 点击菜单栏"Session" → "New Session"
2. 或使用快捷键 Ctrl+N
3. 在弹出窗口中选择"SSH"类型

#### 步骤二：配置连接参数

基本配置：
- Host: 输入服务器地址（如 192.168.1.100 或 example.com）
- Port: SSH 端口（默认 22）
- Username: 登录用户名
- Password: 登录密码（可选）

高级配置：
- Authentication Method: 选择认证方式
- Key File: 指定私钥文件路径
- Proxy: 配置代理设置

#### 步骤三：保存并连接

1. 点击"Save"保存会话配置
2. 点击"Connect"建立连接
3. 首次连接会提示保存服务器指纹

### 密钥认证配置

#### 生成密钥对

1. 点击菜单 "Tools" → "Key Generator"
2. 选择密钥类型（RSA/ED25519/ECDSA）
3. 设置密钥长度和密码短语
4. 点击"Generate"生成密钥
5. 保存公钥和私钥文件

#### 配置密钥认证

1. 在会话属性中选择 "Authentication"
2. 选择 "Public Key" 认证方式
3. 指定私钥文件路径
4. 输入密钥密码短语（如设置了的话）

### SFTP 文件传输

#### 打开 SFTP 面板

1. 建立 SSH 连接后
2. 点击底部的"SFTP"标签页
3. 或使用快捷键 Ctrl+Shift+S

#### 传输文件

上传文件：
- 从本地面板拖拽文件到远程面板
- 右键文件选择"Upload"

下载文件：
- 从远程面板拖拽文件到本地面板
- 右键文件选择"Download"

#### 批量操作

1. 按住 Ctrl 键选择多个文件
2. 右键选择操作（上传/下载/删除）
3. 或使用"Transfer Queue"查看传输队列

### 终端分屏

#### 水平分屏

1. 右键标签页选择"Split Horizontally"
2. 或使用快捷键 Ctrl+Shift+H

#### 垂直分屏

1. 右键标签页选择"Split Vertically"
2. 或使用快捷键 Ctrl+Shift+V

#### 调整分屏大小

- 拖动分屏边框调整大小
- 双击边框平均分配空间

### 串口连接配置

#### 基本设置

1. 新建会话时选择"Serial"类型
2. 选择 COM 端口
3. 配置串口参数：
   - Baud Rate: 9600/115200 等
   - Data Bits: 8
   - Stop Bits: 1
   - Parity: None
   - Flow Control: None

#### 高级设置

- 行结束符设置（CR/LF/CRLF）
- 终端类型选择
- 回显模式配置

### 端口转发

#### 本地端口转发

用途：通过 SSH 隧道访问远程服务

配置步骤：
1. 打开会话属性 → "Tunnel"
2. 添加本地端口转发规则
3. 设置本地端口和远程目标地址
4. 建立 SSH 连接后自动激活隧道

#### 远程端口转发

用途：将远程端口转发到本地服务

配置步骤：
1. 在"Tunnel"设置中添加远程转发规则
2. 指定远程端口和本地目标地址
3. 建立连接后远程端口开始监听

#### 动态端口转发

用途：创建 SOCKS 代理

配置步骤：
1. 设置动态端口转发
2. 指定本地监听端口
3. 在应用程序中配置 SOCKS 代理

### 配置文件位置

- 配置文件位于程序目录下的 `config` 文件夹

- `%APPDATA%\WindTerm\`

3. 选择导入选项
4. 点击"Import"

## 常见问题

### 1. 连接超时怎么办？

排查步骤：
1. 检查网络连接是否正常
2. 确认服务器地址和端口是否正确
3. 检查防火墙设置
4. 尝试使用 ping 或 telnet 测试连通性
5. 增加连接超时时间

### 2. SSH 密钥认证失败

解决方案：
1. 确认私钥文件路径正确
2. 检查密钥文件权限（应为 600）
3. 确认服务器已添加对应公钥
4. 检查 authorized_keys 文件权限（应为 644）
5. 查看服务器 SSH 日志排查问题

### 3. 中文显示乱码

解决方法：
1. 在会话属性中设置字符编码为 UTF-8
2. 在终端设置中调整字体为支持中文的字体
3. 确认远程服务器的 locale 设置正确

### 4. SFTP 传输速度慢

优化建议：
1. 检查网络带宽
2. 增加 SFTP 缓冲区大小
3. 使用压缩传输
4. 避免传输大量小文件，可先打包

### 5. 终端响应延迟

可能原因及解决：
1. 网络延迟 - 使用更近的服务器
2. DNS 解析慢 - 使用 IP 地址连接
3. 服务器负载高 - 联系管理员
4. 终端缓冲区过大 - 调整缓冲区设置

### 6. 如何设置代理连接？

配置步骤：
1. 打开会话属性 → "Proxy"
2. 选择代理类型（SOCKS4/SOCKS5/HTTP）
3. 输入代理服务器地址和端口
4. 如需认证，输入用户名和密码
5. 保存并重新连接

### 7. 便携版和安装版有什么区别？

| 特性 | 便携版 | 安装版 |
|------|--------|--------|
| 安装过程 | 无需安装 | 需要安装 |
| 配置存储 | 程序目录 | AppData 目录 |
| 系统集成 | 无 | 注册文件关联 |
| 移动性 | 可放U盘使用 | 固定安装 |
| 升级方式 | 替换文件 | 覆盖安装 |

### 8. 如何录制和回放宏？

录制宏：
1. 点击菜单 "Macro" → "Start Recording"
2. 执行要录制的操作
3. 点击 "Macro" → "Stop Recording"
4. 保存宏文件

回放宏：
1. 点击 "Macro" → "Run Macro"
2. 选择宏文件
3. 设置回放选项
4. 开始回放

### 9. 快捷键冲突如何解决？

解决方法：
1. 打开 "Tools" → "Preferences" → "Key Bindings"
2. 搜索冲突的快捷键
3. 修改或删除冲突的绑定
4. 保存设置

### 10. 如何查看连接日志？

启用日志：
1. 在会话属性中启用"Logging"
2. 设置日志文件路径
3. 选择日志记录级别

查看日志：
- 日志文件为纯文本格式
- 可用文本编辑器打开查看
- 包含完整的会话交互记录
