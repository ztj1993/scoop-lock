# Notepad++ 使用指南

## 简介

Notepad++ 是一个免费的源代码编辑器和记事本替代品，支持多种编程语言和自然语言。

**版本**: 8.7.4  
**官网**: https://notepad-plus-plus.org/  
**许可证**: GPL-3.0-or-later

## 主要功能

- 语法高亮显示
- 代码折叠
- 自动完成
- 多文档编辑
- 正则表达式搜索替换
- 宏录制和回放
- 插件支持
- 多语言界面

## 常用操作

### 文件操作

```
新建文件: Ctrl+N
打开文件: Ctrl+O
保存文件: Ctrl+S
另存为: Ctrl+Shift+S
关闭文件: Ctrl+W
打印: Ctrl+P
```

### 编辑操作

```
撤销: Ctrl+Z
重做: Ctrl+Y
剪切: Ctrl+X
复制: Ctrl+C
粘贴: Ctrl+V
删除: Del
全选: Ctrl+A
查找: Ctrl+F
替换: Ctrl+H
转到行: Ctrl+G
```

### 视图操作

```
放大: Ctrl++
缩小: Ctrl+-
重置缩放: Ctrl+0
显示行号: 视图 → 显示符号 → 显示行号
显示空白字符: 视图 → 显示符号 → 显示空白字符
显示换行符: 视图 → 显示符号 → 显示换行符
```

### 代码编辑

```
注释: Ctrl+Q
取消注释: Ctrl+Shift+Q
缩进: Tab
取消缩进: Shift+Tab
自动完成: Ctrl+Space
函数列表: 视图 → 函数列表
```

## 搜索和替换

### 基本搜索

```
查找: Ctrl+F
查找下一个: F3
查找上一个: Shift+F3
快速查找: Ctrl+F3
```

### 高级搜索

```
在文件中查找: Ctrl+Shift+F
在文件中替换: Ctrl+Shift+H
```

### 正则表达式搜索

```
打开搜索对话框: Ctrl+F
勾选"正则表达式"
使用正则表达式语法
```

常用正则表达式示例:
- `.` - 匹配任意字符
- `*` - 匹配前一个字符零次或多次
- `+` - 匹配前一个字符一次或多次
- `?` - 匹配前一个字符零次或一次
- `\d` - 匹配数字
- `\w` - 匹配字母、数字、下划线
- `\s` - 匹配空白字符
- `^` - 匹配行首
- `$` - 匹配行尾
- `[]` - 匹配括号内的任意字符
- `()` - 分组

### 标签页操作

```
切换标签页: Ctrl+Tab
下一个标签页: Ctrl+PageDown
上一个标签页: Ctrl+PageUp
关闭标签页: Ctrl+W
关闭所有标签页: Ctrl+Shift+W
```

### 分屏编辑

```
水平分屏: 视图 → 移动/复制当前文档 → 水平分屏
垂直分屏: 视图 → 移动/复制当前文档 → 垂直分屏
```

## 快捷键自定义

### 修改快捷键

```
设置 → 快捷键映射
选择要修改的命令
点击修改
按下新的快捷键
```

### 常用自定义快捷键

```
F5: 运行外部程序
F6: 编译
F7: 构建
Ctrl+D: 复制当前行
Ctrl+L: 删除当前行
Ctrl+Shift+Up: 上移当前行
Ctrl+Shift+Down: 下移当前行
```

## 运行外部程序

### 配置外部工具

```
运行 → 运行
输入命令行
点击保存
```

### 常用外部工具配置

```
# 打开当前文件所在目录
explorer $(CURRENT_DIRECTORY)

# 用浏览器打开当前文件
$(FULL_CURRENT_PATH)

# 运行 Python 脚本
python $(FULL_CURRENT_PATH)

# 运行 Node.js 脚本
node $(FULL_CURRENT_PATH)

# 编译 C++ 文件
g++ $(FULL_CURRENT_PATH) -o $(NAME_PART).exe
```

## 常见问题

### 1. 如何设置默认编码？

```
设置 → 首选项 → 新建
设置默认编码
```

### 2. 如何显示行号？

```
视图 → 显示符号 → 显示行号
```

### 3. 如何自动保存？

```
安装 AutoSave 插件
或
设置 → 首选项 → 备份 → 启用自动保存
```

### 4. 如何比较文件？

```
安装 Compare 插件
插件 → Compare → Compare
```

### 5. 如何录制和回放宏？

```
1. 宏 → 开始录制
2. 执行要录制的操作
3. 宏 → 停止录制
4. 宏 → 保存录制
5. 宏 → 回放
```

## 高级功能

### 正则表达式替换示例

```
# 删除空行
查找: ^\r?\n
替换: (留空)

# 在每行末尾添加分号
查找: $
替换: ;

# 删除行首空格
查找: ^\s+
替换: (留空)

# 提取邮箱地址
查找: \b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b
替换: $0
```

### 批量操作

```
# 批量重命名文件
1. 打开多个文件
2. 使用列编辑模式
3. 编辑文件名

# 批量修改代码
1. 使用正则表达式替换
2. 在文件中替换功能
```
