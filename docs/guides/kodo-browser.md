# Kodo Browser 使用指南

## 简介

Kodo Browser（七牛浏览器）是七牛云提供的图形化对象存储管理工具，用于管理和操作七牛云
对象存储（Kodo）中的资源。Kodo Browser v2.0.1 采用 Apache-2.0 开源许可证，
源代码托管在 GitHub 上。

七牛云对象存储（Kodo）是七牛云推出的大规模分布式存储平台，适用于各类非结构化数据的存储，
包括图片、音视频、文档、日志等。Kodo Browser 提供了直观的图形化界面，让用户可以像
使用 Windows 资源管理器一样方便地管理云端文件。

Kodo Browser 支持基本的文件操作（上传、下载、删除、复制、移动），也支持批量操作、
断点续传、大文件分片上传等功能。它是七牛云用户的日常管理利器，特别适合不熟悉命令行
或 API 的用户使用。

- **版本**: v2.0.1
- **许可证**: Apache-2.0
- **官方网站**: https://github.com/qiniu/kodo-browser
- **安装方式**: Scoop 包管理器

## 主要功能

### 存储空间管理

Kodo Browser 支持对七牛云存储空间（Bucket）进行全面管理：

- **创建空间**: 创建新的存储空间，选择存储区域和访问权限
- **查看空间列表**: 浏览账号下所有存储空间
- **空间设置**: 修改空间的访问权限、缓存策略等设置
- **空间概览**: 查看空间的存储用量、文件数量等统计信息
- **删除空间**: 删除不再使用的存储空间

### 文件管理

提供类资源管理器的文件操作体验：

- **上传文件**: 支持单文件和批量文件上传
- **下载文件**: 支持单文件和批量文件下载
- **浏览文件**: 以列表或缩略图模式浏览存储空间中的文件
- **新建文件夹**: 在存储空间中创建文件夹（虚拟目录）
- **删除文件**: 删除单个或批量删除文件
- **复制/移动文件**: 在同一空间或不同空间之间复制/移动文件
- **重命名**: 修改文件名称
- **查看属性**: 查看文件大小、类型、最后修改时间、ETag 等信息

### 大文件上传

支持大文件的分片上传：

- 自动分片：大文件自动拆分为多个分片上传
- 断点续传：网络中断后可从断点继续上传
- 并发上传：多个分片同时上传，提高传输速度
- 进度显示：实时显示上传进度和速度

### 批量操作

- 批量上传：选择多个文件或整个文件夹上传
- 批量下载：选择多个文件批量下载
- 批量删除：选择多个文件批量删除
- 批量复制/移动：多文件批量操作
- 全选/反选：方便的文件选择操作

### URL 管理

- 生成文件的外链地址
- 复制文件 URL 到剪贴板
- 预览图片文件
- 设置 URL 有效期
- 支持自定义域名的 URL 生成

### 多账号管理

- 支持添加多个七牛云账号
- 快速切换不同账号
- 支持 AccessKey/SecretKey 认证
- 支持多种身份验证方式

### 高级功能

- **文件预览**: 支持图片、音视频文件的在线预览
- **文件搜索**: 按文件名搜索存储空间中的文件
- **前缀过滤**: 按文件前缀过滤显示
- **目录同步**: 支持本地目录与云端目录的同步
- **传输管理**: 查看上传/下载队列和历史记录
- **CDN 刷新**: 触发 CDN 缓存刷新

## 常用命令/操作

### 启动工具

Scoop 安装后直接运行：

```bash
kodo-browser
```

或通过开始菜单和桌面快捷方式启动。

### 账号配置

首次启动时需要配置七牛云账号：

1. 打开 Kodo Browser
2. 点击 `新建账号` 或 `添加账号`
3. 输入以下信息：
   - **账号名称**: 便于识别的别名
   - **Access Key**: 七牛云 AccessKey（在七牛云控制台获取）
   - **Secret Key**: 七牛云 SecretKey
   - **区域**: 选择存储空间所在的区域
4. 点击 `确定` 保存

获取 AccessKey 和 SecretKey：
1. 登录七牛云控制台（https://portal.qiniu.com）
2. 进入 `个人中心` → `密钥管理`
3. 查看或创建 AccessKey/SecretKey

### 存储空间操作

**查看存储空间列表：**

登录后，左侧栏将显示该账号下所有可用的存储空间。点击空间名称即可浏览其中的文件。

**创建新存储空间：**

1. 点击工具栏上的 `新建空间` 按钮
2. 输入空间名称
3. 选择存储区域（华东、华北、华南、北美等）
4. 选择访问权限：
   - **公开空间**: 文件可通过 URL 直接访问
   - **私有空间**: 文件访问需要签名验证
5. 点击 `确定` 创建

**切换存储空间：**

在左侧栏的空间列表中点击不同的空间名称即可切换。

### 文件操作

**上传文件：**

1. 确保当前在目标存储空间中
2. 点击工具栏的 `上传` 按钮，或直接将文件拖拽到窗口中
3. 选择要上传的文件或文件夹
4. 设置目标路径（可选）
5. 点击 `开始上传`
6. 在传输队列中查看上传进度

**下载文件：**

1. 在文件列表中选择要下载的文件（支持多选）
2. 点击工具栏的 `下载` 按钮
3. 选择本地保存路径
4. 下载开始后可在传输队列中查看进度

**删除文件：**

1. 选择要删除的文件
2. 点击 `删除` 按钮或按 Delete 键
3. 确认删除操作

**复制/移动文件：**

1. 选择源文件
2. 点击 `复制` 或 `移动` 按钮
3. 选择目标存储空间和路径
4. 确认操作

**新建文件夹：**

1. 点击 `新建文件夹` 按钮
2. 输入文件夹名称
3. 按 Enter 确认

**文件重命名：**

1. 右键点击文件
2. 选择 `重命名`
3. 输入新名称
4. 按 Enter 确认

### URL 操作

**复制文件 URL：**

1. 右键点击文件
2. 选择 `复制链接` 或 `复制外链`
3. 链接已复制到剪贴板，可直接粘贴使用

**预览文件：**

1. 双击支持预览的文件（图片、文本等）
2. 图片将在新窗口中显示预览

### 传输管理

查看传输队列：

1. 点击底部的 `传输列表` 标签
2. 查看正在上传/下载的任务
3. 可以暂停、继续或取消传输任务

查看传输历史：

1. 切换到 `已完成` 标签
2. 查看历史传输记录

### 设置与偏好

打开设置面板：

1. 点击菜单 `设置` 或工具栏齿轮图标
2. 常用配置项：
   - **并发上传数**: 同时上传的文件数量（默认 5）
   - **分片大小**: 大文件分片大小（默认 4MB）
   - **上传限速**: 限制上传带宽
   - **下载限速**: 限制下载带宽
   - **自动重试**: 传输失败时自动重试
   - **默认编码**: 文件名编码设置
   - **代理设置**: 配置网络代理

## 常见问题

### Q: 无法连接到七牛云账号？

排查步骤：
1. 检查 AccessKey 和 SecretKey 是否正确
2. 检查网络连接是否正常
3. 确认账号没有被禁用
4. 检查是否需要配置代理

### Q: 上传速度慢怎么办？

1. 增加并发上传数（设置 → 并发数）
2. 检查本地网络带宽
3. 选择距离最近的存储区域
4. 配置代理（如果网络受限）
5. 减小分片大小以适应不稳定网络

### Q: 大文件上传失败怎么办？

1. 检查分片大小设置是否合理
2. 开启断点续传功能
3. 确保存储空间有足够的容量
4. 检查单次上传限制
5. 尝试减小并发数

### Q: 如何批量操作文件？

1. 按住 Ctrl 键点击多个文件进行多选
2. 按住 Shift 键选择连续的文件范围
3. 使用 Ctrl+A 全选当前目录下的所有文件
4. 选中后点击工具栏的批量操作按钮

### Q: 私有空间的文件如何分享？

私有空间的文件需要生成带签名的临时 URL：
1. 右键点击文件
2. 选择 `复制签名链接`
3. 链接包含临时签名，可在有效期内访问
4. 可在设置中配置 URL 有效期

### Q: 如何更新 Kodo Browser？

使用 Scoop 更新：

```bash
scoop update kodo-browser
```

或在工具中检查更新。

### Q: 支持哪些文件类型的预览？

- 图片：JPG、PNG、GIF、BMP、WebP 等
- 文本：TXT、JSON、XML、CSV、Markdown 等
- 音视频：MP3、MP4 等（部分格式）
- 其他格式可能需要下载后查看

### Q: 如何设置文件的访问权限？

存储空间级别：
1. 在七牛云控制台中修改空间权限（公开/私有）

文件级别：
1. 通过 CDN 配置访问控制
2. 使用签名 URL 控制临时访问
