# DiskGenius 使用指南

## 简介

DiskGenius 是一款功能强大的磁盘管理与数据恢复工具，当前版本为 v5.6.1.1580。

DiskGenius 集磁盘分区管理、数据恢复、备份还原、系统迁移等多种功能于一体，是国内最流行的磁盘工具之一。它支持各种存储设备，包括机械硬盘、固态硬盘、U 盘、存储卡等，是维护电脑和恢复数据的必备工具。

**主要特性：**
- 全中文界面，操作简便
- 数据恢复功能强大
- 分区管理功能完善
- 支持多种文件系统
- 支持各种存储设备
- 备份还原功能可靠

**官方网站：** https://www.diskgenius.com

## 主要功能

### 1. 数据恢复
DiskGenius 提供专业的数据恢复功能，支持删除文件恢复、格式化恢复、分区丢失恢复、RAID 恢复等多种场景。能够恢复各种类型的文件，包括文档、照片、视频、音频等。

### 2. 分区管理
提供完整的分区管理功能，包括创建分区、删除分区、调整分区大小、合并分区、拆分分区、转换分区格式等。支持无损调整分区大小。

### 3. 磁盘备份与还原
支持整个磁盘或分区的备份与还原，可以创建磁盘镜像文件。支持增量备份，节省存储空间。

### 4. 系统迁移
可以将操作系统从一个磁盘迁移到另一个磁盘，例如从 HDD 迁移到 SSD，无需重新安装系统。

### 5. 磁盘检测与修复
提供磁盘健康检测功能，可以检查磁盘坏道、修复文件系统错误。支持 S.M.A.R.T. 信息查看。

### 6. 文件管理
内置文件管理器，可以直接浏览和操作分区中的文件，支持在不同分区间复制、移动文件。

## 常用命令/操作

### 启动与界面

```bash
# 启动 DiskGenius
DiskGenius.exe

# 以管理员身份运行（推荐）
# 右键点击 DiskGenius.exe → 以管理员身份运行
```

**主界面介绍：**

1. **菜单栏** - 包含所有功能菜单
2. **工具栏** - 常用功能快捷按钮
3. **磁盘列表** - 显示所有连接的磁盘
4. **分区信息** - 显示选中磁盘的分区详情
5. **文件浏览** - 浏览分区中的文件
6. **状态栏** - 显示当前操作状态

### 分区管理操作

#### 创建新分区

1. **选择未分配空间**
   - 在磁盘列表中选择目标磁盘
   - 右键点击未分配空间
   - 选择"建立新分区"

2. **设置分区参数**
   - 选择分区类型（主分区/扩展分区）
   - 选择文件系统（NTFS/FAT32/exFAT/ext4 等）
   - 设置分区大小
   - 设置盘符

3. **确认并执行**
   - 点击"确定"确认设置
   - 点击工具栏"保存更改"按钮
   - 等待操作完成

#### 调整分区大小

1. **选择目标分区**
   - 右键点击要调整的分区
   - 选择"调整分区大小"

2. **设置新大小**
   - 拖动分区边界调整大小
   - 或直接输入新的分区大小
   - 选择释放空间的方向

3. **执行调整**
   - 点击"开始"按钮
   - 等待操作完成
   - 可能需要重启电脑

#### 合并分区

1. **选择相邻分区**
   - 右键点击要保留的分区
   - 选择"合并分区"

2. **选择要合并的分区**
   - 勾选要合并的其他分区
   - 选择文件系统类型

3. **确认合并**
   - 注意：合并会删除被合并分区的数据
   - 确认后点击"开始"

#### 删除分区

1. **选择目标分区**
   - 右键点击要删除的分区
   - 选择"删除当前分区"

2. **确认删除**
   - 系统会提示确认
   - 注意：删除分区会丢失所有数据
   - 确认后执行删除

#### 转换分区格式

1. **选择目标分区**
   - 右键点击要转换的分区
   - 选择"格式化"

2. **选择新文件系统**
   - 选择目标文件系统类型
   - 设置簇大小（可选）
   - 设置卷标（可选）

3. **执行格式化**
   - 确认后执行格式化
   - 注意：格式化会清除分区所有数据

### 数据恢复操作

#### 恢复删除的文件

1. **选择目标分区**
   - 在磁盘列表中选择文件所在的分区
   - 点击工具栏"恢复文件"按钮

2. **选择恢复模式**
   - "仅恢复误删除的文件" - 快速扫描
   - "完整恢复" - 深度扫描（推荐）
   - 选择扫描范围

3. **开始扫描**
   - 点击"开始"按钮
   - 等待扫描完成
   - 扫描时间取决于分区大小

4. **预览和选择文件**
   - 在扫描结果中查找目标文件
   - 使用预览功能确认文件完整性
   - 勾选要恢复的文件

5. **执行恢复**
   - 点击"恢复"按钮
   - 选择保存位置（不要保存到原分区）
   - 等待恢复完成

#### 恢复丢失的分区

1. **选择目标磁盘**
   - 选择包含丢失分区的磁盘
   - 点击工具栏"搜索分区"按钮

2. **选择搜索范围**
   - "整个磁盘" - 搜索整块磁盘
   - "指定范围" - 指定搜索的起始和结束位置

3. **开始搜索**
   - 点击"开始搜索"按钮
   - 等待搜索完成

4. **确认找到的分区**
   - 检查找到的分区是否正确
   - 保留正确的分区
   - 继续搜索其他分区

5. **保存分区表**
   - 点击"保存更改"
   - 重启电脑验证

#### 恢复格式化的分区

1. **选择已格式化的分区**
   - 选择目标分区
   - 点击"恢复文件"

2. **选择完整恢复**
   - 勾选"额外扫描已知文件类型"
   - 这会进行深度扫描

3. **等待扫描完成**
   - 深度扫描需要较长时间
   - 可以暂停和继续扫描

4. **恢复文件**
   - 选择要恢复的文件
   - 保存到其他位置

### 备份与还原

#### 备份分区

1. **选择要备份的分区**
   - 右键点击目标分区
   - 选择"备份分区到镜像文件"

2. **设置备份选项**
   - 选择镜像文件保存位置
   - 选择压缩级别
   - 设置分卷大小（可选）

3. **开始备份**
   - 点击"开始"按钮
   - 等待备份完成

#### 还原分区

1. **选择镜像文件**
   - 点击工具栏"还原分区"按钮
   - 选择之前创建的镜像文件

2. **选择目标分区**
   - 选择要还原到的分区
   - 注意：还原会覆盖目标分区数据

3. **开始还原**
   - 点击"开始"按钮
   - 等待还原完成

#### 克隆磁盘

1. **选择源磁盘**
   - 点击工具栏"克隆磁盘"按钮
   - 选择源磁盘

2. **选择目标磁盘**
   - 选择目标磁盘
   - 注意：目标磁盘会被完全覆盖

3. **设置克隆选项**
   - 选择是否按文件复制（更快）
   - 选择是否调整分区大小

4. **开始克隆**
   - 点击"开始"按钮
   - 等待克隆完成

### 磁盘检测

#### 检查磁盘坏道

1. **选择目标磁盘**
   - 右键点击要检测的磁盘
   - 选择"坏道检测与修复"

2. **设置检测范围**
   - 选择检测整个磁盘
   - 或指定起始和结束位置

3. **开始检测**
   - 点击"开始检测"按钮
   - 等待检测完成
   - 绿色表示正常，红色表示坏道

4. **尝试修复**
   - 如果发现坏道
   - 可以尝试"尝试修复"功能
   - 注意：修复可能导致数据丢失

#### 检查文件系统

1. **选择目标分区**
   - 右键点击要检查的分区
   - 选择"检查分区表错误"

2. **选择检查方式**
   - "自动修复文件系统错误"
   - "扫描并尝试恢复坏扇区"

3. **开始检查**
   - 点击"开始"按钮
   - 等待检查完成

#### 查看 S.M.A.R.T. 信息

1. **选择目标磁盘**
   - 点击菜单"磁盘" → "查看 S.M.A.R.T. 信息"

2. **查看健康状态**
   - 查看各项参数值
   - 关注"健康状态"指示
   - 注意异常警告

### 系统迁移

#### 迁移系统到 SSD

1. **选择源系统盘**
   - 点击工具栏"系统迁移"按钮
   - 选择当前系统所在磁盘

2. **选择目标 SSD**
   - 选择新的 SSD
   - 确保 SSD 容量足够

3. **调整分区大小**
   - 可以调整系统分区大小
   - 选择是否迁移其他分区

4. **开始迁移**
   - 点击"开始"按钮
   - 等待迁移完成
   - 可能需要重启电脑

5. **设置启动**
   - 迁移完成后进入 BIOS
   - 设置 SSD 为第一启动项
   - 验证系统正常启动

## 常见问题

### Q1: 数据恢复成功率有多高？

**影响因素：**

1. **数据丢失原因**
   - 删除文件：成功率较高（90%+）
   - 格式化分区：成功率中等（60-80%）
   - 分区丢失：成功率较高（70-90%）
   - 物理损坏：成功率较低

2. **是否写入新数据**
   - 丢失后立即恢复：成功率最高
   - 继续使用写入数据：成功率下降
   - 多次写入覆盖：可能无法恢复

3. **存储设备类型**
   - HDD 机械硬盘：恢复成功率较高
   - SSD 固态硬盘：由于 TRIM 功能，成功率较低
   - U 盘/存储卡：成功率中等

**提高成功率的建议：**
- 发现数据丢失后立即停止使用该分区
- 不要在原分区上安装恢复软件
- 将恢复的文件保存到其他磁盘

### Q2: 调整分区大小时提示空间不足怎么办？

**解决方案：**

1. **清理目标分区**
   - 删除不需要的文件
   - 清空回收站
   - 清理临时文件

2. **缩小相邻分区**
   - 先缩小相邻分区释放空间
   - 再扩大目标分区

3. **使用未分配空间**
   - 如果有未分配空间
   - 可以直接使用这部分空间

### Q3: 恢复的文件无法打开怎么办？

**排查步骤：**

1. **检查文件完整性**
   - 使用预览功能查看文件
   - 如果预览正常，可能是软件关联问题

2. **尝试其他恢复软件**
   - 不同软件的恢复算法不同
   - 可以尝试其他工具恢复

3. **文件头修复**
   - 部分文件可能文件头损坏
   - 可以尝试专业数据恢复服务

### Q4: 如何恢复 SSD 上的数据？

**SSD 数据恢复的挑战：**

1. **TRIM 功能**
   - SSD 的 TRIM 功能会主动擦除已删除数据
   - 一旦 TRIM 执行，数据几乎无法恢复

2. **提高成功率的方法**
   - 发现数据丢失后立即断开 SSD
   - 不要对 SSD 进行任何写入操作
   - 尽快使用恢复工具扫描

3. **专业恢复**
   - 如果自行恢复失败
   - 可以考虑专业数据恢复服务
   - 费用较高但成功率更有保障

### Q5: 如何创建 WinPE 启动盘？

**创建步骤：**

1. **下载 WinPE 工具**
   - 从微软官方下载 Windows ADK
   - 或使用第三方 WinPE 工具

2. **集成 DiskGenius**
   - 将 DiskGenius 复制到 WinPE
   - 创建启动 U 盘

3. **使用场景**
   - 系统无法启动时进行数据恢复
   - 在 WinPE 环境下进行分区操作
   - 进行系统备份和还原

### Q6: 如何检查硬盘健康状态？

**检查方法：**

1. **查看 S.M.A.R.T. 信息**
   - 菜单 → 磁盘 → 查看 S.M.A.R.T. 信息
   - 关注"重新分配扇区计数"
   - 关注"当前待映射扇区数"

2. **坏道检测**
   - 进行完整的坏道扫描
   - 记录坏道数量和位置
   - 定期监测坏道增长趋势

3. **注意警告信号**
   - 硬盘发出异常声音
   - 系统频繁卡顿
   - 文件读写错误增多

### Q7: DiskGenius 支持哪些文件系统？

**支持的文件系统：**

| 文件系统 | 读写支持 | 说明 |
|---------|---------|------|
| NTFS | 完全支持 | Windows 主要文件系统 |
| FAT32 | 完全支持 | 兼容性最好 |
| exFAT | 完全支持 | 大容量U盘推荐 |
| EXT2/3/4 | 完全支持 | Linux 文件系统 |
| HFS/HFS+ | 只读支持 | macOS 文件系统 |
| APFS | 只读支持 | 新版 macOS 文件系统 |
| ReFS | 只读支持 | Windows Server 文件系统 |

### Q8: 如何处理"分区表错误"提示？

**解决方案：**

1. **备份重要数据**
   - 立即备份分区中的重要文件
   - 分区表错误可能导致数据丢失

2. **检查并修复**
   - 使用"检查分区表错误"功能
   - 让 DiskGenius 自动修复

3. **重建分区表**
   - 如果自动修复失败
   - 可以尝试搜索丢失的分区
   - 重建分区表

4. **专业帮助**
   - 如果不确定如何操作
   - 建议寻求专业人士帮助
   - 错误操作可能导致数据永久丢失
