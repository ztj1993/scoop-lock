# NyaTerm 使用指南

## 简介

NyaTerm 是一个现代化的远程终端工作区，使用 Tauri、React 和 Rust 构建。

**版本**: 1.0.9  
**官网**: https://nyaterm.app/  
**许可证**: MIT  
**类型**: 图形界面应用程序  
**可执行文件**: NyaTerm.exe

## 主要功能

- 现代化用户界面
- 多标签页终端
- SSH 连接管理
- 主题自定义
- 快捷键支持
- 会话管理
- 数据持久化

## 常用操作

### 启动应用

```
开始菜单 → NyaTerm
或
双击 NyaTerm.exe
```

### 基本操作

```
新建连接: Ctrl+N
关闭标签: Ctrl+W
切换标签: Ctrl+Tab
复制: Ctrl+C
粘贴: Ctrl+V
全屏: F11
```

### SSH 连接

```
1. 点击"新建连接"
2. 输入主机地址
3. 输入端口（默认 22）
4. 输入用户名
5. 选择认证方式（密码/密钥）
6. 点击"连接"
```

### 会话管理

```
保存会话: 自动保存
加载会话: 从会话列表选择
删除会话: 右键 → 删除
```

## 配置选项

### 外观设置

```
字体: 设置终端字体
字号: 设置字体大小
主题: 选择颜色主题
透明度: 设置窗口透明度
```

### 快捷键设置

```
自定义快捷键: 设置 → 快捷键
重置快捷键: 恢复默认设置
```

## 常见问题

### 1. 无法连接到服务器

```
可能原因:
- 网络不通
- SSH 服务未启动
- 防火墙阻止

解决方法:
1. 检查网络连接
2. 确认 SSH 服务状态
3. 检查防火墙规则
```

### 2. 中文显示乱码

```
解决方法:
1. 设置终端编码为 UTF-8
2. 检查服务器语言设置
```
