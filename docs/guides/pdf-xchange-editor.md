## 简介

PDF-XChange Editor 是一款功能强大且轻量级的 PDF 查看和编辑工具，由 Tracker Software Products 开发。它提供了丰富的 PDF 处理功能，包括文本编辑、注释、表单填写、OCR 识别、数字签名等，是 Adobe Acrobat 的优秀替代品。

PDF-XChange Editor 以其快速的启动速度、低内存占用和丰富的功能而著称。免费版本提供了大量实用功能，付费版本则解锁了更多高级特性。它是目前市场上最受欢迎的轻量级 PDF 编辑器之一。

### 核心特性

- **快速启动**：启动速度快，内存占用低
- **文本编辑**：直接编辑 PDF 中的文本内容
- **注释工具**：丰富的注释和标记工具
- **表单处理**：填写和创建 PDF 表单
- **OCR 识别**：将扫描文档转换为可搜索文本
- **数字签名**：支持 PDF 数字签名
- **页面管理**：合并、拆分、旋转、删除页面
- **水印功能**：添加文字和图片水印
- **安全设置**：密码保护和权限控制
- **批量处理**：支持批量操作

## 常用操作

### 启动应用

```powershell
# 从开始菜单或桌面快捷方式启动
# 或从 Scoop 安装目录运行
& "$env:USERPROFILE\scoop\apps\pdf-xchange-editor\current\PDFXEdit.exe"

# 直接打开指定 PDF 文件
& "$env:USERPROFILE\scoop\apps\pdf-xchange-editor\current\PDFXEdit.exe" "C:\path\to\document.pdf"
```

### 基本查看操作

#### 打开文件

- 菜单：文件 → 打开
- 快捷键：`Ctrl+O`
- 直接拖拽文件到窗口

#### 页面导航

| 操作 | 方法 |
|------|------|
| 上一页 | Page Up 或 ↑ 键 |
| 下一页 | Page Down 或 ↓ 键 |
| 第一页 | Home 键 |
| 最后一页 | End 键 |
| 跳转到指定页 | `Ctrl+G`，输入页码 |
| 使用缩略图 | 点击左侧缩略图面板中的页面 |

#### 缩放操作

| 操作 | 方法 |
|------|------|
| 放大 | `Ctrl++` 或 Ctrl+滚轮上 |
| 缩小 | `Ctrl+-` 或 Ctrl+滚轮下 |
| 适合页面 | `Ctrl+0` |
| 适合宽度 | `Ctrl+2` |
| 实际大小 | `Ctrl+1` |
| 自定义缩放 | 状态栏缩放百分比 |

### 注释操作

#### 高亮文本

1. 选择「注释」工具栏中的「高亮」工具
2. 用鼠标选中要高亮的文本
3. 高亮颜色可以在属性中修改

#### 添加便签

1. 选择「注释」工具栏中的「便签」工具
2. 点击要添加注释的位置
3. 在弹出的窗口中输入注释内容

#### 添加文本框

1. 选择「注释」工具栏中的「文本框」工具
2. 在页面上拖拽创建文本框
3. 输入文本内容
4. 可以调整字体、大小、颜色等属性

#### 图章工具

1. 选择「注释」工具栏中的「图章」工具
2. 选择预设图章或自定义图章
3. 在页面上放置图章

#### 图形标注

1. 选择「注释」工具栏中的图形工具（矩形、圆形、箭头等）
2. 在页面上拖拽绘制图形
3. 可以设置线条颜色、粗细、填充颜色等

### 文本编辑（付费功能）

#### 编辑现有文本

1. 选择「编辑」工具栏中的「编辑文本」工具
2. 点击要编辑的文本
3. 修改文本内容
4. 调整格式设置

#### 添加新文本

1. 选择「编辑」工具栏中的「添加文本」工具
2. 点击要添加文本的位置
3. 输入文本内容
4. 设置字体、大小、颜色等

#### 查找和替换

- 查找：`Ctrl+F`
- 替换：`Ctrl+H`
- 支持正则表达式查找

### 表单操作

#### 填写表单

1. 打开包含表单的 PDF
2. 点击表单字段进行填写
3. 使用 Tab 键在字段间切换
4. 填写完成后保存

#### 保存表单数据

- 保存为 PDF：保留表单数据
- 导出表单数据：文件 → 导出表单数据

### 页面管理

#### 插入页面

1. 菜单：页面 → 插入页面
2. 选择插入位置
3. 选择页面来源（空白、文件等）

#### 删除页面

1. 选择要删除的页面（缩略图面板）
2. 右键 → 删除页面
3. 或使用菜单：页面 → 删除页面

#### 旋转页面

1. 选择要旋转的页面
2. 右键 → 旋转页面
3. 选择旋转方向（顺时针/逆时针）

#### 合并文档

1. 菜单：文件 → 新建文档 → 合并文件
2. 添加要合并的 PDF 文件
3. 调整文件顺序
4. 点击确定

#### 拆分文档

1. 菜单：页面 → 拆分文档
2. 设置拆分规则（按页数、按大小等）
3. 选择输出位置
4. 点击确定

### OCR 文字识别（付费功能）

#### 执行 OCR

1. 打开扫描的 PDF 文档
2. 菜单：文档 → OCR 页面
3. 选择识别语言
4. 设置输出选项
5. 点击确定

#### OCR 设置

- 语言选择：支持多国语言
- 输出模式：可搜索文本或可编辑文本
- 页面范围：全部页面或指定页面
- 精度设置：平衡速度和准确度

### 数字签名（付费功能）

#### 添加签名

1. 菜单：签名 → 添加签名
2. 选择签名方式：
   - 绘制签名
   - 输入签名
   - 使用数字证书
3. 在文档中放置签名
4. 保存签名文档

#### 验证签名

- 签名状态会在文档中显示
- 可以查看签名详情和证书信息

### 安全设置

#### 设置密码保护

1. 菜单：文件 → 文档属性 → 安全
2. 设置打开密码
3. 设置权限密码
4. 配置打印和编辑权限
5. 保存设置

#### 密文（Redaction）

1. 选择「保护」工具栏中的「密文」工具
2. 标记要密文处理的文本或区域
3. 应用密文
4. 确认删除内容

### 批量处理

#### 批量转换

1. 菜单：文件 → 批量处理
2. 添加要处理的文件
3. 选择操作类型
4. 设置输出选项
5. 执行处理

## 常见问题

### Q1: PDF-XChange Editor 免费版有什么限制？

免费版的限制包括：

- 创建的 PDF 会有水印
- 无法使用文本编辑功能
- 无法使用 OCR 功能
- 无法使用数字签名
- 部分高级功能受限

但免费版仍提供了丰富的查看、注释和表单填写功能，足以满足大部分日常使用需求。

### Q2: 如何去除创建 PDF 时的水印？

去除水印需要购买付费许可证。或者：

1. 使用免费的在线 PDF 创建工具
2. 使用其他免费 PDF 虚拟打印机
3. 购买 PDF-XChange Editor Plus 许可证

### Q3: 如何提高 OCR 识别准确度？

1. **选择正确的语言**：确保选择了文档中实际使用的语言
2. **提高扫描质量**：使用 300 DPI 或更高的扫描分辨率
3. **确保文档清晰**：模糊或倾斜的文档会影响识别效果
4. **预处理文档**：先进行图像增强处理
5. **校对结果**：OCR 完成后手动校对关键内容

### Q4: 如何批量处理多个 PDF 文件？

1. 使用菜单中的「批量处理」功能
2. 添加多个文件到处理列表
3. 选择要执行的操作
4. 设置输出选项
5. 执行批量处理

### Q5: PDF 文件过大如何压缩？

1. 菜单：文件 → 另存为优化大小
2. 选择压缩选项：
   - 图像压缩质量
   - 删除嵌入字体
   - 删除元数据
3. 保存优化后的文件

### Q6: 如何在 PDF 中添加水印？

1. 菜单：页面 → 水印 → 添加
2. 选择水印类型：
   - 文字水印：输入文字内容
   - 图片水印：选择图片文件
3. 设置水印样式：
   - 位置、角度、透明度
   - 字体、大小、颜色
4. 选择应用的页面范围
5. 确认添加

### Q7: 如何自定义工具栏？

1. 菜单：视图 → 工具栏 → 自定义
2. 可以：
   - 显示/隐藏工具栏
   - 重排工具栏位置
   - 添加/移除工具按钮
   - 创建自定义工具栏
